Security Headers

Checks CSP, HSTS, X-Frame-Options, X-Content-Type-Options, Referrer-Policy, and Permissions-Policy.

🍪

Cookie Security

Detects missing HttpOnly and Secure flags, and warns about sensitive cookie exposure.

🔀

Mixed Content

Finds HTTP resources loaded on HTTPS pages — scripts, styles, images, and iframes.

📝

Form Security

Validates HTTPS submission, CSRF token presence, and proper password field handling.

📚

Vulnerable Libraries

Detects outdated jQuery, Angular, Lodash, Bootstrap, Moment.js and reports CVEs automatically.

Sensitive Data Exposure
